# KIXIMA Website MVP

Protótipo funcional em Next.js com o visual corporativo KIXIMA e fluxos centrais do marketplace.

## Como executar

```bash
npm install
npm run dev
```

Abrir: http://localhost:3000

## Páginas incluídas

- `/` — Landing com seleção dos módulos
- `/comprador` — Persona Comprador
- `/fornecedor` — Persona Fornecedor
- `/financeiro` — Persona Financeiro simplificado para e-market
- `/admin` — Company Admin
- `/fluxos` — Fluxo geral do Marketplace KIXIMA

## Princípio do MVP

Este website mantém a KIXIMA como e-market/procurement garantido, não como ERP. O foco está em catálogo, POs, faturas, pagamentos, entregas, aprovações, conformidade e dashboards essenciais.
